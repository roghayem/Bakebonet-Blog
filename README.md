## 🍰 Bakebonet – Baking Blog

A clean, modern, and responsive baking blog showcasing sweets, desserts, and handmade recipes — built with HTML and CSS. Perfect for demonstrating front-end design skills, responsive layouts, and creative UI.

## ✨ Features

⚡ Fully responsive design for desktop, tablet, and mobile

📝 Contact form layout with clean design (without JS functionality)

🌐 Integrated social media icons (Twitter, Instagram, Email)

🎨 Modern UI design with gradients, shadows, and rounded elements

📚 Sections for blog posts and recipes with visual thumbnails



---

## 🛠️ Technologies Used

HTML5

CSS3 (Flexbox, Grid, Animations, Transitions, Gradients)

Google Fonts (e.g., Poppins)

Font Awesome for icons
